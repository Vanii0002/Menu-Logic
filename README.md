# Option C – Menu Logic

This project is part of the **Web Developer Internship Skill Assessment** at Larviz Digital.

## Features
- Display menu items from JSON
- Add items to cart
- Calculate total price
- Apply **10% discount** automatically if total > ₹500
- Log final order object on checkout

## Tech Stack
- HTML, CSS
- JavaScript (Vanilla)
- Axios
